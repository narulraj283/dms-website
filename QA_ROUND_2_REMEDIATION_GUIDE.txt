ISSUE: GA4 Tracking (63 Files) - Missing window.dataLayer initialization
STATUS: CRITICAL - Prevents analytics data collection
IMPACT: Google Analytics 4 is not collecting data properly

SOLUTION:
All 63 files need the same fix. Add one line to their GA4 scripts.

STEPS:

1. Locate GA4 Script Section
   Find this section in the file's <head>:

   <!-- Google Analytics -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-Z7DFTL44YL"></script>
   <script>
     function gtag(){dataLayer.push(arguments);}
     gtag('js',new Date());
     gtag('config','G-Z7DFTL44YL');
   </script>

2. Add Missing Line
   Change it to:

   <!-- Google Analytics -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-Z7DFTL44YL"></script>
   <script>
     window.dataLayer = window.dataLayer || [];
     function gtag(){dataLayer.push(arguments);}
     gtag('js',new Date());
     gtag('config','G-Z7DFTL44YL');
   </script>

3. The Critical Line Added:
   window.dataLayer = window.dataLayer || [];

SCRIPT EXPLANATION:
- <script async src="...">  : Loads gtag.js library asynchronously
- window.dataLayer = ...    : MISSING LINE - Initializes data array
- function gtag()           : Defines function to push data
- gtag('js', new Date())    : Initializes gtag with timestamp
- gtag('config', 'G-...')   : Configures measurement ID

FILES REQUIRING THIS FIX (53 files with incomplete GA4):
- /tmp/dms-repo/404.html
- /tmp/dms-repo/index.html
- 25 files in /tmp/dms-repo/pages/
- 26 files in /tmp/dms-repo/articles/
- 6 files in /tmp/dms-repo/tools/

VERIFICATION:
After editing, search each file for "window.dataLayer = window.dataLayer"
If found, the fix is complete for that file.

AUTOMATED FIX (Python):
You can use this Python script to fix all files automatically:

```python
import glob
from pathlib import Path

pattern = "/tmp/dms-repo/**/*.html"
files_to_fix = glob.glob(pattern, recursive=True)

for filepath in files_to_fix:
    with open(filepath, 'r', encoding='utf-8') as f:
        content = f.read()

    # Skip if already has the line
    if "window.dataLayer = window.dataLayer" in content:
        continue

    # Skip if no GA4 tracking at all
    if "G-Z7DFTL44YL" not in content:
        continue

    # Find and replace
    old_pattern = (
        '<script>\n  function gtag(){dataLayer.push(arguments);}'
    )
    new_pattern = (
        '<script>\n  window.dataLayer = window.dataLayer || [];\n'
        '  function gtag(){dataLayer.push(arguments);}'
    )

    if old_pattern in content:
        new_content = content.replace(old_pattern, new_pattern)
        with open(filepath, 'w', encoding='utf-8') as f:
            f.write(new_content)
        print(f"Fixed: {filepath}")
```

TESTING:
1. Open a website page in Chrome
2. Press F12 to open DevTools
3. Go to Console tab
4. Type: window.dataLayer
5. Should return an array, not undefined

Or write a script to update all files at once:

```python
from pathlib import Path
import re

repo_path = Path("/tmp/dms-repo")

for html_file in repo_path.glob("**/*.html"):
    content = html_file.read_text(encoding='utf-8')

    # Replace relative paths in href attributes
    # Change "pages/X.html" to "/pages/X.html"
    updated = re.sub(
        r'href=["\'](?<!/)([^/"]+/[^"\']+)["\']',
        r'href="/\1"',
        content
    )

    # Change "../pages/X.html" to "/pages/X.html"
    updated = re.sub(
        r'href=["\']\.\./(pages/[^"\']+)["\']',
        r'href="/\1"',
        updated
    )

    if updated != content:
        html_file.write_text(updated, encoding='utf-8')
        print(f"Updated: {html_file}")
```

```python
import re

ACTUAL_DOMAIN = "dentalhq.com"  # Change this to real domain
EMAIL_PREFIX = "support"  # Change if different

repo_path = Path("/tmp/dms-repo/email-templates")

for html_file in repo_path.glob("*.html"):
    content = html_file.read_text(encoding='utf-8')

    # Replace domain
    updated = content.replace(
        "dentalhq.example.com",
        ACTUAL_DOMAIN
    )

    # Replace email
    updated = updated.replace(
        f"support@dentalhq.example.com",
        f"{EMAIL_PREFIX}@{ACTUAL_DOMAIN}"
    )

    if updated != content:
        html_file.write_text(updated, encoding='utf-8')
        print(f"Updated: {html_file}")
```
