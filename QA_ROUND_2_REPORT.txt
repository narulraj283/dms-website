1. GA4 TRACKING VERIFICATION - 63 ISSUES CRITICAL
================================================================================

SEVERITY: CRITICAL
IMPACT: Analytics data collection is incomplete

ISSUE DESCRIPTION:
All 63 files have the gtag.js script and measurement ID (G-Z7DFTL44YL), but are
missing the critical "window.dataLayer = window.dataLayer || [];" initialization
line. This line is REQUIRED for proper GA4 data collection.

CURRENT IMPLEMENTATION (INCORRECT):
---
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-Z7DFTL44YL"></script>
<script>
  function gtag(){dataLayer.push(arguments);}
  gtag('js',new Date());
  gtag('config','G-Z7DFTL44YL');
</script>
---

REQUIRED IMPLEMENTATION:
---
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-Z7DFTL44YL"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js',new Date());
  gtag('config','G-Z7DFTL44YL');
</script>
---

AFFECTED FILES (63 total):
- /tmp/dms-repo/404.html
- /tmp/dms-repo/index.html
- /tmp/dms-repo/articles/* (26 files)
- /tmp/dms-repo/pages/* (25 files)
- /tmp/dms-repo/tools/* (6 files)
- /tmp/dms-repo/pages/dashboard-admin.html (no GA4 at all)
- /tmp/dms-repo/pages/dashboard-member.html (no GA4 at all)
- /tmp/dms-repo/assets/* (2 files - no GA4)
- /tmp/dms-repo/email-templates/* (7 files - no GA4)
- /tmp/dms-repo/includes/author-bio.html (no GA4)

RECOMMENDATION:
Add the missing window.dataLayer initialization line to all tracked files.
Consider creating a global header include file to ensure consistency.
